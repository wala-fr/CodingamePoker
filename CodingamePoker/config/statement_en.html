Let's play poker !
This is a port of the Texas hold'em version.
The source code of this game is available on this GitHub repository .

  RULES

The rules of Texas hold 'em can be found all over the web . For example : wikipedia with the hand ranking .

Game inputs
The number of players can be from 2 to 4 .
The total of all players buy-ins is $4800 . So for 2 players the buy-in is $2400 , for 3 players it's $1600 and for 4 players it's $1200 .
The value of the small blind / big blind are $5 / 10$ .
Then every 10 hands, the level is increased and the blinds (small and big) are multiplied by a 2.
The first big blind player is random.
These values are given as game inputs.
The action moves clockwise around the table. The player 0 is at the top (in heads up he's on the left ).
The game ends when :
  • one player owns all the chips.
  • after 600 rounds (if the last hand is not over, it's cancelled ).
The players are ranked by elimination round and stack.
The card's abbreviations are 2D , KH ... ( complete list ).
A list of how some special cases are handled is here .

  ACTIONS

Possible actions

There are 5 possible actions :
  • CALL
  • CHECK
  • BET amount , the amount is the number of chips that the player adds to the pot.
  • ALL-IN
  • FOLD

Action replacement

In some cases, if you give an "invalid" action, it will be automatically replaced by a "legal" one.
For example :
  • BET 500 with a stack of $400 will be considered as an ALL-IN .
  • TOTO or other invalid action will be considered as a FOLD .
The complete list can be found here .
However if you FOLD when you could just CHECK , it won't be replaced.
The replacement informations are written in the output console.

  BLINDS

The small blind and big blind initial values are given in the initial inputs.
Then every " number of hands by level " the level is increased and the blinds (small and big) are multiplied by a coefficient (those values are also given in the initial inputs).
Victory Conditions
  • You own all the chips.
  • After 600 rounds you own the biggest stack.
Loss Conditions
  • You've lost all your chips.
  • You do not respond in time.

  Game Input

Initialization input
Line 1 : startSmallBlind
Line 2 : startBigBlind the initial values of the blinds.
Line 3 : handlNbByLevel the number of hands before the level increases.
Line 4 : levelBlindMultiplicator the blinds are multiplied by this coefficient when the level changes.
Line 5 : playerNb the initial number of players.
Line 6 : playerId your id.

Input for one game turn
Line 1 : round referee round (game will end when it reaches 600 ).
Line 2 : pot number of chips in the pot.
Next playerNb lines : stack stack of each player.
Next line : boardCardNb number of board cards already dealt.
Next boardCardNb lines : boardCard board card abbreviation (2H, KD... see abreviations ).

Output
A single line containing the player action.
Example: BET 200
You can also add a message :
Example: FOLD;MESSAGE
Constraints
2 ≤ number of players ≤ 4
Response time first turn is ≤ 1000 ms.
Response time per turn is ≤ 50 ms.
round ≤ 600
